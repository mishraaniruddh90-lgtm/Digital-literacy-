Project Overview

This repository contains my Digital Literacy Portfolio created as part of the CSE0001 Digital Literacy course. As a Digital Ambassador , I have prepared a digital portofolio of mine using various digital tools. The purpose of this project is to develop essential digital skills such as creating a professional online presence, understanding digital communication, using collaboration platforms, practicing coding on online platforms, and learning about cybercrime awareness and online safety.

This project is divided into five tasks, each focusing on a different aspect of digital literacy. All the tasks, screenshots, documents, and the final project report are organized into separate folders in this repository.

Tasks Included Task 1 – Digital Literacy Infographic

Created a digital literacy awareness infographic using an online canvas and explained the importance of digital literacy, safe internet practices, and professional online presence.

Task 2 – Student Digital Portfolio

Created and updated professional profiles on platforms like GitHub and LinkedIn to build a digital portfolio for academic and career growth.

Task 3 – Coding & Collaboration Platforms
